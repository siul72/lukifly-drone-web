Introduction

GPSM is to be used under MORPHEUS platforms to provide an interface to LEA6-R Ublox GPS device.
It abstracts the communication between the application who need GPS data,
Provides IO velocity-related data by creating an easy and fast interface to all data.
It also as the functionality of date and time synchronization over GPS data.
